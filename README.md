# Generative AI Research Project

**This is a research project repository on "Generative AI Tools".**


## Project structure

**Literature_Review** : Stores periodicals, books and news articles.
**Quantitative_Analysis** : Stores survey data, Python scripts and analysis reports.
**Qualitative_Analysis**: Store interview records, consent forms, and visual charts.
**Drafts_and_Reports**: Stores draft research proposals and final reports.
**Media**: Stores photos and other media files.
